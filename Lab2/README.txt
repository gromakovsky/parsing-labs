В папке Parser лежат исходники парсера. Исполняемый файл читает input.txt и выводит граф в формате .dot в стандартный output. Для этого используется Boost Graph Library (нужен Boost, чтобы собралось).
Класс parser принимает при конструировании поток входных данных, а его метод parse() парсит эти данные и возвращает дерево.
В папке Testing есть файл tests, в котором на каждой строке находится тест. Скрипты run_tests и make_graph делают то, что сказано в их названии (запуская Parser и dot).
В папке Report есть отчёт (report.pdf) и исходники для него.
